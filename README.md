# Ultimate One Night Werewolf Helper (終極一夜狼人殺語音助手)

這是一個純前端的網頁應用程式，旨在作為「終極一夜狼人殺」遊戲的語音主持人。玩家可以透過此工具選擇參與的角色、設定人數，並在遊戲開始後由系統自動播放語音引導流程。

## 📋 專案功能

- **角色自定義**：支援選擇不同的角色組合（如：狼人、爪牙、預言家、強盜、搗蛋鬼、失眠者等）。
- **流程自動化**：根據選擇的角色自動生成對應的夜間行動語音指令。
- **語音引導**：使用 Web Speech API 或音檔播放，提供清晰的遊戲主持語音。
- **倒數計時**：提供白天討論時間的倒數功能。
- **響應式設計**：支援手機與平板瀏覽，方便在桌遊聚會中使用。
- **GitHub Pages 部署**：無需後端伺服器，直接透過靜態網頁提供服務。

## 🛠 技術棧

- **前端框架**：Vanilla JS (或考慮 React/Vue，初期以純 JS 為主以保持輕量)
- **樣式**：Tailwind CSS (透過 CDN 或建構工具)
- **語音技術**：Web Speech API (Text-to-Speech)
- **部署**：GitHub Pages
- **版本控制**：Git

## 🚀 遊戲流程邏輯

1. **配置階段**：
   - 玩家選擇本局遊戲包含的角色。
   - 確認玩家人數與牌堆總數（人數 + 3）。
2. **準備階段**：
   - 確認所有玩家已就緒。
3. **夜間階段 (語音主持)**：
   - 「所有人閉上眼睛。」
   - 依照角色順序依序喚醒玩家並倒數。
   - 「所有人睜開眼睛。」
4. **討論階段**：
   - 開始白天討論倒數。

## 🎭 角色資訊與夜間順序

本專案支援《終極一夜狼人殺》及其擴充包（如：曙光 Dawn、吸血鬼 Vampire 等）中的角色。

### 夜間行動順序表

| 順序 | 角色                                  | 行動描述                                                       | 關鍵細節與爭議釐清                                                                                                                                                               |
| :--- | :------------------------------------ | :------------------------------------------------------------- | :------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 1    | **化身幽靈 (Doppelganger)**           | 查看一名玩家的牌並變為該角色。                                 | 若變為有夜間行動的角色，**立即**執行該行動。若變為狼人或守夜人，則加入該陣營且在後續對應順序睜眼與夥伴相認。若變為失眠者，則不可在失眠者睜眼時同時睜眼，應在化身失眠者階段才睜眼。 |
| 2    | **狼人 (Werewolf)**                   | 狼人睜開眼，確認彼此身份。                                     | 此時**所有**狼人身分角色（包含純狼人、狼王、神秘狼、化身狼人）應同時睜眼相認。                                                                                                   |
| 3    | **狼王 (Alpha Wolf)**                 | 睜開眼，將中央額外設置 (垂直) 的一張狼人牌與另一名玩家交換。   | 此行動在全體狼人相認後單獨進行。不能查看交換後的牌。                                                                                                                             |
| 4    | **神秘狼 (Mystic Wolf)**              | 睜開眼，查看一名其他玩家的牌。                                 | 此行動在全體狼人相認後單獨進行。                                                                                                                                                 |
| 5    | **爪牙 (Minion)**                     | 爪牙睜開眼，狼人舉起大拇指讓爪牙確認。                         | 狼人不睜眼。所有狼人身分（含狼王、神秘狼）皆需舉手。爪牙若沒看到狼人，則判定狼人在中央牌。                                                                                       |
| 6    | **守夜人 (Mason)**                    | 守夜人們睜開眼，確認彼此身份。                                 | 包含化身守夜人。                                                                                                                                                                 |
| 7    | **預言家 (Seer)**                     | 預言家睜開眼，可查看一名玩家的牌，或查看中央兩張牌。           | 不能同時看一名玩家又看一張中央牌。                                                                                                                                               |
| 8    | **學徒預言家 (Apprentice Seer)**      | 睜開眼，查看中央的一張牌。                                     | 僅限中央牌。                                                                                                                                                                     |
| 9    | **強盜 (Robber)**                     | 強盜睜開眼，可選擇交換自己與另一名玩家的牌，並查看自己的新牌。 | 強盜交換後即變為新身分，但本局仍以強盜順序行動。                                                                                                                                 |
| 10   | **搗蛋鬼 (Troublemaker)**             | 搗蛋鬼睜開眼，可交換另外兩名玩家的牌。                         | 不能看牌，不能選自己。                                                                                                                                                           |
| 11   | **酒鬼 (Drunk)**                      | 酒鬼睜開眼，將自己的牌與中央的一張牌交換。                     | **不准**看牌。酒鬼自此不知道自己的新身分。                                                                                                                                       |
| 12   | **失眠者 (Insomniac)**                | 失眠者睜開眼，查看自己目前的牌。                               | 確認自己的牌是否被強盜或搗蛋鬼動過。                                                                                                                                             |
| 13   | **化身失眠者 (Apprentice Insomniac)** | 只有當化身幽靈變成失眠者時行動。                               | 邏輯同失眠者。                                                                                                                                                                   |

### 夜間不行動角色表 (村民、特殊職能)

這些角色在夜間階段不睜眼，但其技能或勝利條件影響討論與結算。

| 角色 | 描述與關鍵細節 |
| :--- | :--- |
| **村民 (Villager)** | 無特殊能力，屬於正義陣營。 |
| **獵人 (Hunter)** | 若被投票處死，他指向的一名玩家也同時死亡。 |
| **皮匠 (Tanner)** | 厭世者，只有自己被投票處死時才獨自獲勝。 |
| **守衛 (Bodyguard)** | 投票時指向的玩家獲得保護，即便該玩家票數最高也不會死亡（改由票數第二高者死亡）。 |
| **王子 (Prince)** | 無論票數多高都不會被處死。 |

## ⚖️ 投票結算與平局規則 (Voting & Tiebreaker Rules)

在討論時間結束後，所有玩家應同時伸出手指向一名玩家。根據投票結果，將決定誰被處死以及最終的陣營勝負。

1. 死亡判定準則
   - 多數決：獲得最高票且票數 > 2 票 的玩家被處死。
   - 平局判定：若有多名玩家同時獲得最高票（且皆 > 2 票），則所有最高票者同時死亡。
   - 全體平票 (1 票)：若所有玩家都只獲得 1 票，則無人死亡。這代表玩家集體決定「這場遊戲沒有狼人」。
2. 陣營勝負判定

遊戲的勝負將在翻牌並確認死者身份後判定：

| 死亡情況                     | 勝負結果     |
| :--------------------------- | :----------- |
| 死者中包含至少一名狼人       | 村民陣營獲勝 |
| 死者中不包含任何狼人         | 狼人陣營獲勝 |
| 無人死亡，且場上真的沒有狼人 | 村民陣營獲勝 |
| 無人死亡，但場上其實有狼人   | 狼人陣營獲勝 |

3. 特殊角色對平局的影響

當特定角色捲入平局或投票時，結算邏輯如下：

🛡 王子 (Prince)
- 免疫死亡：若王子獲得最高票（無論是獨得或平局），他不會死亡。
- 連帶影響：在平局情況下，若王子與一名狼人同為最高票，則王子存活，僅有狼人死亡（村民陣營獲勝）。

🏹 獵人 (Hunter)
- 臨死反擊：若獵人在投票中死亡（包含平局死亡），他所指向的那名玩家也會立即死亡。
- 最終結算：判定勝負時，會將「被獵人帶走的人」一併納入死者名單計算。

🧤 皮匠 (Tanner)
- 獨立勝負：皮匠只有在自己死亡時才會獲勝。
- 特殊優先權：若皮匠死亡，且死者中不包含狼人，則只有皮匠單獨獲勝；若皮匠與狼人同時在平局中死亡，則皮匠與村民陣營共同獲勝。
